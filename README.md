# Tuk-Tuk Real-Time Detection System

An IoT-based tuk-tuk security system designed to improve vehicle safety and prevent theft, vandalism, and unauthorized access. The system uses ESP8266 Wi-Fi module, magnetic door sensor, flame sensor, load cell, relay module, and the Blynk mobile application for real-time monitoring and alerts.

## Features
- Real-time alerts through Blynk app
- Unauthorized seat access detection
- Fire detection using flame sensor
- Seat occupancy monitoring using load cell
- Remote vehicle activation using relay module
- Low-cost and user-friendly security solution

## Technologies Used
- ESP8266 Wi-Fi Module
- Arduino IDE
- Blynk IoT Platform
- LM35 / Flame Sensor / Magnetic Sensor
- Relay Module
- Load Cell Sensor
